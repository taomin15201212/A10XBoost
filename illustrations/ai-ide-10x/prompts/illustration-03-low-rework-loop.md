少返工开发闭环 - Process Flow

Layout: circular flowchart with clockwise direction

STEPS:
1. 明确边界 - 目标 / 约束 / 验收
2. 先计划后编码 - 模块拆分 / 文件变更 / 风险与回滚
3. 写完自查 - 逻辑说明 / 冒烟+边界测试 / 运行命令 / 常见坑
4. 带证据修复 - 终端报错 / 日志 / 文件片段 -> 最小修改 -> 回归建议

CONNECTIONS:
- Bold arrows clockwise between 4 steps
- Feedback arrow from step 4 back to step 1 labeled "更新约束与验收"
- Each step has Input and Output mini-tags

SUPPORTING LABELS:
- 主标题: 把效率做上去的关键：少返工闭环
- 副标题: 从“生成代码”到“可交付代码”

STYLE: notion flowchart style, clear node hierarchy, minimal decorative elements, strong readability for Chinese text
ASPECT: 16:9, medium complexity
