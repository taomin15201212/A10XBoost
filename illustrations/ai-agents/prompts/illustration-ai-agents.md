# Article Illustration Prompts

## Style Guide: Notion Style

Notion style characteristics:
- Minimalist hand-drawn line art
- Monochrome or limited color palette (black/gray on white/off-white)
- Imperfect, sketchy lines that look human-drawn
- Clean, uncluttered compositions
- Geometric shapes with hand-drawn quality
- Light, airy feel
- Professional but approachable

## Illustration 1: Copilot vs Agent Workflow Comparison

**Filename**: 01-flowchart-copilot-vs-agent.png
**Aspect Ratio**: 16:9

### Prompt

A split-screen comparison diagram in minimalist hand-drawn line art style (Notion style).

LEFT SIDE - "Copilot Mode" (labeled in Chinese "Copilot模式"):
- A loop diagram showing: User → Command Input → AI Response → User Execution → (back to start)
- Use circular arrows to indicate repetition
- Show 2-3 cycles to emphasize the loop
- Small stick figure user at the start
- Speech bubble icons for commands
- Robot head icon for AI
- Hand/click icon for user execution

RIGHT SIDE - "Agent Mode" (labeled in Chinese "Agent模式"):
- A linear flow: User Goal → AI Decomposition → Auto Execution → Deliver Result
- Single arrow flowing downward
- User stick figure at top
- Robot icon in the middle with "task breakdown" gears
- Checkmark box at bottom for completed result
- Emphasize the one-time closed loop with a bold enclosing box

Style notes:
- Black lines on off-white background
- Hand-drawn imperfect lines
- Simple geometric icons
- Chinese labels in clean sans-serif
- Arrows are sketchy and organic
- Some lines can be slightly wobbly

- Organic, slightly wobbly connector lines

## Illustration 5: 7-Day Rollout Timeline

**Filename**: 05-timeline-7day-rollout.png
**Aspect Ratio**: 16:9

### Prompt

A horizontal Gantt-style timeline in minimalist hand-drawn line art style (Notion style).

HEADER: "7天落地路径" (7-Day Rollout Path)

TIMELINE STRUCTURE:
Horizontal bar divided into 4 phases.

PHASE 1 (Days 1-2): "环境准备" (Environment Setup)
- Short bar segment
- Icon: gear/wrench
- Subtasks listed below:
  * 确定场景 (Define scenario)
  * 选择工具 (Select tools)
  * 搭建环境 (Setup environment)

PHASE 2 (Days 3-4): "首个Agent" (First Agent)
- Medium bar segment
- Icon: robot/rocket
- Subtasks:
  * 单任务Agent (Single-task agent)
  * 接入外部工具 (External tools)
  * 测试记录 (Test & log)

PHASE 3 (Days 5-6): "工程化" (Engineering)
- Medium bar segment
- Icon: tools/wrench
- Subtasks:
  * 观测工具 (Observability)
  * 错误处理 (Error handling)
  * 安全配置 (Security config)

PHASE 4 (Day 7): "复盘迭代" (Review & Iterate)
- Short bar segment
- Icon: chart/graph
- Subtasks:
  * 统计数据 (Statistics)
  * 决策优化 (Decision)

Style:
- Horizontal timeline bar across the image
- Different shades of gray for each phase
- Hand-drawn segment dividers
- Simple icons above each phase
- Small text labels for subtasks
- Organic, sketchy bars (not perfect rectangles)
- Milestone markers at day boundaries
- Light background, black/gray lines
