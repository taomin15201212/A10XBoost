# illustrations 插图资源目录

## 目录职责

本目录包含"AI编程从0到1之10X提效"项目所有篇章的插图资源，按模块分类存储，支持文章内容的可视化表达。

## 子目录结构

```
illustrations/
├── ai-ide-10x/                    # 01 IDE编辑器插图
├── ai-llm-vendors-models/         # 02 LLM编码模型插图
├── ai-cli-tools-top10/            # 03 CLI终端工具插图
├── ai-extensions-plugins/         # 04 扩展插件插图
├── ai-agents/                     # 05-A Agents智能体插图
├── ai-workflow-orchestration/     # 05-B AI工作流编排插图
├── agent-skills/                  # 06 Agent Skills插图
├── mcp-protocol-guide/            # 07 MCP协议插图
├── vibe-design-08/                # 08 Vibe Design插图
└── vibe-coding-09/                # 09 Vibe Coding插图
```

## 插图命名规范

- `01-` 开头: 对比图 (comparison)
- `02-` 开头: 框架图 (framework)
- `03-` 开头: 流程图 (flowchart)
- `04-` 开头: 时间线 (timeline)
- `05-` 开头: 信息图 (infographic)

## 文件格式

- 主文件: `.png` (通用兼容)
- 优化版本: `.webp` (Web使用)
- 备份文件: `*-backup-YYYYMMDD-HHMMSS.png`
